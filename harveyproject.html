Bank

Bank is the largest JAVA project I have done using Hashsets and ArrayLists, this is my final project for my ZeroToInternShip course, in this project there are five classes: BankInterface, Bank, Account, Transactions, and User. BankInterface creates banks. In the Bank class home page there are 10 options (Methods) for the user to choose to execute; Create Account, Delete Account, Transfer to , Deposit, Withdraw, Show Acc details, Delete User, Log out, Exit, Spend. The entire Bank is set up as Run the program --> show home page --> Register as an User --> Go to User home page --> log in to your Account --> choose from any of the 10 options above --> ex. create a new Account(Checking/Saving) this account will go under this user, a user can have maximum 5 accounts. I used a HastSet so whenever a User is created it is linked with an ArrayList called Accounts, there can be maximum 5 accounts, within each account there is another ArrayList Transactions that tranks every transactions made with this account.
